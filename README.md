# Webová aplikace notářské kanceláře

Tento projekt je ročníkovou prací a jeho cílem je vytvoření moderní webové aplikace pro notářskou kancelář. Aplikace umožňuje klientům získat informace o službách, ceníku, kontaktech a galerii. 
Registrovaní uživatelé si mohou rezervovat termíny online a notářka je o nových rezervacích automaticky informována. Součástí je také administrační rozhraní pro správu obsahu.

---

## Funkce aplikace

- Přehled služeb, ceníku a galerie kanceláře  
- Registrace a přihlášení klientů  
- Rezervace termínů přes interaktivní kalendář  
- Automatická notifikace notářce o nové rezervaci  
- Potvrzení rezervace zaslané klientovi e-mailem  
- Sekce FAQ (často kladené dotazy)  
- Admin rozhraní pro správu obsahu webu  
- Responzivní design pro všechna zařízení  
- Bezpečnostní prvky proti spamování a ochrana databáze  
- Funkce resetu hesla  
- Návrh budoucího chatbota pro zefektivnění komunikace

---
